# Web-id Portfolio

## Overview
A static portfolio website for ELFAR DEV, built with plain HTML, CSS, and JavaScript. Features include particle effects, dark/light theme toggle, glitch text animations, and a responsive layout.

## Project Structure
- `index.html` - Main HTML page (SEO optimized)
- `style.css` - All styles including theme support
- `script.js` - Interactive features (particles, theme toggle, navigation, etc.)
- `server.js` - Simple Node.js static file server (port 5000)
- Image assets: `IMG_2997.jpeg`, `IMG_2998.jpeg`, `azalea-view.jpg`, `webktlg.jpg`, `webprofile.jpg`

## Running
The project uses a Node.js HTTP server to serve static files on port 5000.
